# Curses_crt.sh
Console app that allows to scrape crt.sh.
### Required libraries
 * requests
 * BeautifulSoup4
 * curses / windows-curses
### How to install
##### Windows
`$ python3 -m pip install -r requirements_windows.txt`
##### Linux
`$ python3 -m pip install -r requirements_linux.txt`
### How to run
`$ python3 main.py`
